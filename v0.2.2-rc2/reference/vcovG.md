# Generalized Covariance (ANHECOVA)

Generalized Covariance (ANHECOVA)

## Usage

``` r
vcovG(x, decompose = TRUE, ...)
```

## Arguments

- x:

  (`prediction_cf`) Counter-factual prediction.

- decompose:

  (`flag`) whether to use decompose method to calculate the variance.

- ...:

  Not used.

## Value

Named covariance matrix.
