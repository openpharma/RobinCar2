# Log Hazard Ratio Coefficient Matrix

This function creates a coefficient matrix for the log hazard ratio
estimates.

## Usage

``` r
h_log_hr_coef_mat(x)
```

## Arguments

- x:

  (`list`) A list containing the log hazard ratio estimates and their
  standard errors.

## Value

A matrix with columns for the log hazard ratio estimate, standard error,
z-value, and p-value.
